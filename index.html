The first framework implementing the Document Oriented Approach.

SimplOn is still incomplete and have lacked active development lately, so it needs to be updated to use the modern PHP standards.

Still, SimplOn represents the first attempt to implement the Document Oriented Approach in a programming framework, to demonstrate DOA advantages.

If you wish to help, please contact us.
